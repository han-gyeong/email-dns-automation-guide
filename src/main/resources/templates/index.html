DNS 이메일 점검

이메일 발송 DNS 점검

도메인과 발송 서버 IP를 입력하면 SPF, DKIM, DMARC, PTR 설정을 자동으로 분석하고 설정 가이드를 생성합니다.

이메일 발송에 사용하는 도메인을 입력하세요.

이메일 발송 서버의 공인 IP를 쉼표(,)로 구분하여 입력하세요.

DKIM 레코드 조회에 사용할 셀렉터 이름입니다. (예: sender, mail, dkim)

점검 항목:  SPF (발송 서버 IP 인증)  ·  DKIM (이메일 서명 키 생성)  ·  DMARC (이메일 정책 설정)  ·  PTR (역방향 DNS 조회)
